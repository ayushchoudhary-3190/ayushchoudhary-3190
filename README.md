#### [distributed-file-system](https://github.com/ayushchoudhary-3190/distributed-file-system)
Fault-tolerant distributed file storage in Go. Parallel chunk uploads using goroutines, consistent hashing for replication, gRPC inter-service communication, deployed on Kubernetes + AWS.
